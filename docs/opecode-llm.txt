# OpenCode Plugin Development

Curated OpenCode documentation for building plugins, custom tools, and supporting agent workflows in JavaScript/TypeScript, including core docs, examples, ecosystem references, and local notes.

## Official docs
- [OpenCode plugins](https://opencode.ai/docs/plugins/): Core plugin concepts, load order, local/global plugin directories, npm plugin loading, hooks, events, compacting hooks, logging, and custom tools inside plugins. [web:41]
- [OpenCode custom tools](https://opencode.ai/docs/custom-tools/): How to build standalone custom tools, define schemas, export multiple tools, and use tool execution context. [web:45]
- [OpenCode agents](https://opencode.ai/docs/agents/): Agent and subagent behavior, delegation model, and where plugin-aware agents fit into orchestration. [page:1]
- [OpenCode skills](https://opencode.ai/docs/skills/): Skill discovery, `SKILL.md` structure, and lazy-loading guidance for reusable agent instructions. [web:11]
- [OpenCode ecosystem](https://opencode.ai/docs/ecosystem/): Community plugins, plugin templates, orchestration tooling, and examples of real-world OpenCode extensions. [page:2]
- [OpenCode home](https://opencode.ai/docs/): Main documentation entrypoint. [web:9]

## Examples
